# Time Simulator Plugin


<img width="1383" src="src/timeSimulatorTopImg.png">


## このプラグインについて

-  現在時刻に任意時刻を設定するプラグインです。
 　
### 操作方法

- タイムスライダーの「つまみ」を掴んでスライドし時刻を変更します。

  ![](src/img1.png)

- 時刻表時部分に直接数字を入力して時刻を変更することも可能です。

  ![](src/img2.png)

## 備考
